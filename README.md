# Violin Rental Agency - Database Application
CPSC 3660 Project

## Tech Stack
- PHP 8.3
- MySQL 8.0
- Apache2 (WSL/Ubuntu)

## Pages
- insert.php — Add a new customer
- update.php — Update an existing customer
- delete.php — Delete a customer
- query.php  — Search and view all customers

## How to Run
1. Start Apache and MySQL in WSL:
   sudo service apache2 start
   sudo service mysql start
2. Open browser and go to: http://localhost/violin-rental/query.php
